# Style Patterns

## Purpose

Use this file to choose a tone pattern quickly and keep the reply sharp without falling into lazy abuse.

## Core Patterns

### 1. 拆逻辑

Use when the target is confidently wrong, internally inconsistent, or hiding behind posture.

Formula:
`点出漏洞` + `压低可信度`

Examples:

- 你这段话最大的作用，是把漏洞一次性展示完整了。
- 结论冲得挺快，理由还没跟上。
- 你不是把话说清了，你是把问题绕开了。

### 2. 假客气

Use when you want the line to look calm on the surface and mean underneath.

Formula:
`礼貌开头` + `冷评价`

Examples:

- 你先别急着下判断，常识还没到场。
- 理解你想显得笃定，可惜内容没跟上气势。
- 这份自信很稳定，依据倒是依旧缺席。

### 3. 镜像反弹

Use when the target uses a memorable phrase or repeated structure.

Formula:
`借对方句式` + `反向落点`

Examples:

- 你不是在输出观点，你是在现场直播漏洞。
- 你不是在反驳别人，你是在证明自己没听懂。
- 你这不是态度强硬，是内容单薄只能靠音量补。

### 4. 冷面终结

Use when a full debate is not worth it and the user wants a clean finish.

Formula:
`低情绪` + `短封口`

Examples:

- 行，先把道理补上再说。
- 你这句适合留给自己反思。
- 先有内容，再来摆架子。

### 5. 降身位

Use when the target pretends to be authoritative or superior.

Formula:
`拆掉权威感` + `改写成噪音`

Examples:

- 口气挺高，论证还在楼下。
- 你抬高的是声调，不是水平。
- 架子搭得很快，内容还是空的。

## Intensity Knobs

Make a line harsher by changing structure, not by adding uglier words.

- Remove explanation and keep only the last twist.
- Replace direct judgment with a colder observation.
- Reuse one keyword from the target so the hit feels tailored.
- Shorten the sentence until it sounds like a verdict.

## Anti-Patterns

Avoid these failure modes:

- Pure profanity with no angle
- Attacks on appearance or identity
- Long moral speeches pretending to be comebacks
- Generic internet filler that fits any fight
- Overwritten sarcasm with too many clauses

## Quick Mapping

Map common situations to a pattern fast:

- `嘴硬但没论据` -> `拆逻辑`
- `装懂/摆谱` -> `降身位`
- `情绪化输出` -> `冷面终结`
- `爱用漂亮句式压人` -> `镜像反弹`
- `想要体面但很损` -> `假客气`
